# Offroad Driving Simulator (In Progress)

A stylized and realistic offroad driving experience built with Unity, featuring advanced vehicle physics and dynamic camera systems.

## Project Status

Currently in active development. Core vehicle physics and camera system are functional.

## Features

- **Realistic Vehicle Physics**: WheelCollider-based 4WD suspension system with customizable handling
- **Advanced Camera System**: Cinemachine-powered dynamic third-person camera
- **Professional Structure**: Organized asset hierarchy and clean C# scripts

## Technical Stack

- **Engine**: Unity 6000.3.5f2 
- **Physics**: WheelCollider suspension system
- **Camera**: Cinemachine
- **Language**: C#

## Requirements

- Unity 6000.3.5f2 (tested with this version, may work with similar LTS releases)
- Cinemachine package (included in project)

## How to Use

1. Open the project in Unity 2022+
2. Load the `Prototype1` scene from Assets/Scenes/
3. Press Play to test

## Controls

- **WASD** - Movement
- **Space** - Handbrake
- **Mouse** - Camera control
